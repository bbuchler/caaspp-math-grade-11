# CAASPP Math Grade 11 Course Package

This folder is the first pilot package for the reusable Education Course Studio.

## Contents

- `source-lock.md` - trusted sources, videos, and open questions
- `course-brief.md` - course purpose and constraints
- `course-map.md` - Week 1 lesson sequence and objective map
- `lessons/` - five student-facing lesson JSON files
- `assessments/` - item bank and open-response rubrics
- `media/` - video plan and infographic specs
- `ai/` - Study Buddy and grading policy
- `assets/` - course branding assets
- `preview.html` - local student-facing preview
- `teacher-dashboard.html` - teacher gradebook preview with 20 lesson columns and Lesson 1 active
- `teacher-student.html` - Lesson 1 work sample, grading, reset, and print view
- `supabase/` - database setup files for the live pilot
- `teacher-notes/` - review notes for Baptiste/teacher use
- `qa-report.md` - readiness report and known risks

## Current State

Week 1 is drafted as a teacher-review pilot. It is ready to inspect as curriculum, but not yet deployed as a working student app.

The preview includes the Altus logo, embedded video frame areas with fallback links, first-pass infographic images, a floating Study Buddy, and local pilot saving for Lesson 1 answers.

## One Thing To Understand

This package fixes the biggest weakness in the original Week 1 workbook: it turns five repeated equation days into a real CAASPP-prep progression across equations, inequalities, representations, slope/intercepts, and function notation.
