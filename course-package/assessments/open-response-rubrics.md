# Open Response Rubrics

## factual-short-2

Use for short answers where the student can phrase the idea in multiple correct ways and partial credit is useful.

| Score | Description |
| ---: | --- |
| 2 | The answer communicates the required mathematical idea, even if wording is informal. |
| 1 | The answer has an important correct piece but misses a meaningful detail, sign, unit, or vocabulary connection. |
| 0 | The answer is blank, off-topic, or shows a misconception about the required idea. |

Teacher review if the answer is hard to interpret or appears to use the right keywords with the wrong meaning.

## math-explanation-3

Use for short explanations of a math idea.

| Score | Description |
| ---: | --- |
| 3 | Correct math idea, clear reasoning, and useful vocabulary or context. |
| 2 | Mostly correct reasoning but missing precision, vocabulary, units, or one small detail. |
| 1 | Partial idea shown, but reasoning is unclear or contains a meaningful misconception. |
| 0 | Blank, off-topic, copied prompt, or no evidence of the target skill. |

Teacher review if the score is 1 or lower, or if the response seems correct but uses unusual reasoning.

## error-analysis-3

Use when students explain another student's mistake.

| Score | Description |
| ---: | --- |
| 3 | Identifies the exact error, explains why it is wrong, and gives or implies the correct move. |
| 2 | Identifies the general error but explanation is thin or the correction is incomplete. |
| 1 | Notices something is wrong but cannot locate the error or gives a weak correction. |
| 0 | Blank, off-topic, or accepts the incorrect work as correct. |

## math-model-3

Use when students create a simple equation or inequality from context.

| Score | Description |
| ---: | --- |
| 3 | Correct model, variables are meaningful, and inequality/equation symbol matches the situation. |
| 2 | Model is mostly right but missing variable explanation, units, or has a small notation issue. |
| 1 | Some quantities are used correctly but the model does not match the situation. |
| 0 | Blank, off-topic, or no usable model. |

## modeling-explanation-5

Use for mini performance task explanations.

| Score | Description |
| ---: | --- |
| 5 | Correct model, correct computation, clear interpretation, and units/rounding make sense in context. |
| 4 | Correct model and answer with minor missing detail in explanation or units. |
| 3 | Reasonable model but one computational or interpretation error. |
| 2 | Partial setup but major missing step or unclear connection to context. |
| 1 | Attempts the task but the model does not represent the situation. |
| 0 | Blank, off-topic, or no mathematical reasoning. |

Teacher review if AI confidence is low, if rounding is the main issue, or if the answer is borderline between 3 and 4.
