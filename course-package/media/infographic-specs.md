# Infographic Specs

## INF-01 - Equation Undo Card

- Lesson: 01
- Purpose: Help students choose inverse operations in the correct order.
- Format: Two-column process card.
- Required labels: "Operation in equation", "Undo move", "Keep both sides balanced", "Check in original equation".
- Do not include: Extra algebra rules not used in Lesson 01.
- Alt text: A chart showing how addition, subtraction, multiplication, and division are undone when solving equations.

## INF-02 - Inequality Graph Decision Table

- Lesson: 02
- Purpose: Help students decide open/closed point and shading direction.
- Format: Symbol table with small number-line sketches.
- Required labels: "<", "<=", ">", ">=", "open point", "closed point", "shade left", "shade right".
- Alt text: A table matching inequality symbols to number-line graph choices.

## INF-03 - Four Ways To Show A Line

- Lesson: 03
- Purpose: Connect situation, table, equation, and graph.
- Format: Four-panel concept map using the rideshare example.
- Required labels: "Words", "Table", "Equation", "Graph".
- Alt text: Four panels showing the same linear relationship as a situation, table, equation, and graph.

## INF-04 - Slope And Intercept In Context

- Lesson: 04
- Purpose: Show that slope and intercept have meanings, not just numbers.
- Format: Annotated graph of a fundraiser model.
- Required labels: "starting amount", "rate per ticket", "for each 1 ticket", "revenue".
- Alt text: A line graph annotated to show the y-intercept as starting money and slope as dollars per ticket.

## INF-05 - Function Machine

- Lesson: 05
- Purpose: Make function notation feel less intimidating.
- Format: Input-rule-output diagram.
- Required labels: "input", "rule", "output", "f(3) means input 3".
- Alt text: A function machine diagram showing an input number going through a rule to produce an output.
