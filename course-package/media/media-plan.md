# Media Plan

## Media Principle

Media must teach a move students will use immediately. For this pilot, every video has a watch-for prompt and contextual questions below it.

## Required Videos

| Lesson | Video | Placement | Student action |
| --- | --- | --- | --- |
| 01 | Linear equations 1 | After quick start | Answer inverse-operation questions, then solve a similar equation. |
| 02 | Solving and graphing linear inequalities | After quick start | Identify open/closed boundary and shading direction. |
| 03 | Graphing a linear equation | Before table-to-graph practice | Explain why a table helps graph a line. |
| 04 | Finding slope and intercepts from tables | Before slope/intercept practice | Identify change in y over change in x and intercept meaning. |
| 05 | Evaluating discrete functions | Before mini performance task | Identify input/output and explain function notation. |

## Video QA Checklist

- Link opens.
- Video title matches lesson.
- Video is not the only explanation.
- Context questions appear directly below the video.
- Transcript or text fallback is available.
- Lesson can still be completed if the video is unavailable.
