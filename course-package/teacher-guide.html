How This Pilot Works

Module 1 starts with Lesson 1 fully built: overview video, embedded lesson video, contextual questions, practice, exit check, Study Buddy, and gradebook tracking. Lessons 2-20 are visible in the gradebook so the course shape is clear, but only Lesson 1 is active for the first review.

Teacher Flow

  1. Open the gradebook.
  2. Add or enroll a test student.
  3. Preview Lesson 1 from the header or gradebook column.
  4. Have the test student answer Lesson 1 questions.
  5. Open the student's work sample from the gradebook.

What To Check

  • Video plays or opens cleanly in a new tab.
  • Students can answer video questions while watching.
  • Objective math items grade immediately.
  • Written explanations are flagged for rubric/AI review.
  • The gradebook shows Lesson 1 progress without hiding the 20-lesson course shape.

Later Enhancements

Once real student responses exist, the trends area can show missed concepts, small group suggestions, remediation paths, and backup lessons. That should come from saved response data, not static cards.
