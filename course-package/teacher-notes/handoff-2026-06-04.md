# CAASPP Math Handoff - 2026-06-04

## Goal

Build a CAASPP Grade 11 math course that works like the successful Photography / Driver's Ed course pattern:

- Teachers log in to a real gradebook.
- Teachers can add students/teachers, reset passwords, view work samples, print/export, and preview lessons.
- Students log in to a separate student dashboard and complete lessons.
- The course eventually has 4 modules / 20 lessons, but V1 should prove the system with one fully working Lesson 1.
- Lesson 1 should include the overview video, embedded YouTube lesson video, contextual video questions, math practice, exit check, auto-grading, and Study Buddy support.
- The shared Supabase backend must keep course data separated by `course_id`; Driver's Ed, Photography, and CAASPP students must not bleed into each other's gradebooks.

## Important Lesson From Today

The shared Supabase project is fine only if every course feature is course-scoped. The bug today was not Supabase itself; it was the dashboard code pulling from global `profiles`, which made Driver's Ed students appear in CAASPP. Do not repeat that pattern.
