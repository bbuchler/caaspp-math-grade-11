# GitHub Publishing

Recommended repository name:

```text
caaspp-math-grade-11
```

## Publish Steps

Create a new GitHub repository named `caaspp-math-grade-11`, then run:

```powershell
cd "C:\school\CAASPP Math Prep Course - 11th Grade"
git remote add origin https://github.com/YOUR_USERNAME/caaspp-math-grade-11.git
git push -u origin main
```

Replace `YOUR_USERNAME` with the GitHub account or organization that should own the repository.

## GitHub Pages Preview Link

After the repository is pushed, enable GitHub Pages from the `main` branch. The preview link should be:

```text
https://YOUR_USERNAME.github.io/caaspp-math-grade-11/course-package/preview.html
```

## Suggested Visibility

Use private while teacher review is happening. If students will use the GitHub Pages preview directly, switch to public or publish through a school-approved hosting option.
